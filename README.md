Monuments of India

The web app created with HTML, CSS and Bootstrap offers an interactive experience where users can explore monuments by hovering over images. Leveraging Bootstrap for responsive design, the app provides a visually appealing layout. When the cursor is placed on an image, a hover effect triggers, revealing a concise description of the monument. This minimalist yet engaging approach allows users to easily discover information about each monument by simply interacting with the images on the page.
